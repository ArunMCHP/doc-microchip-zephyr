Nordic Standalone STM logging snippet (nordic-log-stm)

Overview

This snippet allows users to build Zephyr with the logging to the Coresight STM stimulus ports. Data is collected in ETR buffer. Data from ETR buffer is decoded and human-readable data is output on the UART.
