Wakeup source type encodings

Wakeup source encoding layout.

Bit[31]: 0 = PIN, 1 = MODULE

For PIN (bit[31] = 0):

  • Bit[7:0] : pin index (0-255)
  • Bit[9:8] : edge type (1 = rising, 2 = falling, 3 = any)

For MODULE (bit[31] = 1):

  • Bit[7:0] : module index (0-255)
