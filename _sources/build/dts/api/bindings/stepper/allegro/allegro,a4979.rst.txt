allegro,a4979
#############

.. sidebar:: Overview

   :Name: ``allegro,a4979``
   :Vendor: :ref:`Allegro DVT <dt_vendor_allegro>`
   :Used in: :zephyr:board-catalog:`List of boards <#compatibles=allegro,a4979>` using
               this compatible
   :Driver: :zephyr_file:`drivers/stepper/allegro/a4979.c`

Description
***********



.. code-block:: none

   Allegro A4979 microstepping stepper motor driver.
   A4979 is a flexible microstepping motor driver with built-in translator for easy operation.
   It is designed to operate bipolar stepper motors in full-, half-, quarter-, and sixteenth-step
   modes.

Examples
********

.. code-block:: dts

   a4979: a4979 {
       status = "okay";
       compatible = "allegro,a4979";
       micro-step-res = <2>;
       reset-gpios = <&gpiod 10 GPIO_ACTIVE_HIGH>;
       en-gpios = <&gpiod 11 GPIO_ACTIVE_HIGH>;
       m0-gpios = <&gpiod 13 0>;
       m1-gpios = <&gpiod 12 0>;
   };
