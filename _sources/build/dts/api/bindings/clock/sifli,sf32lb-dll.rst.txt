sifli,sf32lb-dll
################

.. sidebar:: Overview

   :Name: ``sifli,sf32lb-dll``
   :Vendor: :ref:`SiFli Technologies <dt_vendor_sifli>`
   :Used in: :zephyr:board-catalog:`List of boards <#compatibles=sifli,sf32lb-dll>` using
               this compatible

Description
***********



.. code-block:: none

   SiFli DLL is a digital phase-locked loop peripheral, controlled by the RCC.
   It can be used as a clock source for the system and peripherals.
