st,stm32n6-ic-clock-mux
#######################

.. sidebar:: Overview

   :Name: ``st,stm32n6-ic-clock-mux``
   :Vendor: :ref:`STMicroelectronics <dt_vendor_st>`
   :Used in: :zephyr:board-catalog:`List of boards <#compatibles=st,stm32n6-ic-clock-mux>` using
               this compatible

Description
***********



.. code-block:: none

   STM32N6 Divider IC multiplexer.

   This node selects a clock input and a divider.

   For instance:
     &ic6 {
       pll-src = <2>;
       ic-div = <16>;
       status = "okay";
     };
