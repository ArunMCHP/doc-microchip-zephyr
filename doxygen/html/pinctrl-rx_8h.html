Detailed Description

Renesas RX pin control (pinctrl) definitions for Zephyr.

This header provides macro constants for encoding pin function selections and pin indices for Renesas RX SoCs. These values are used by the DeviceTree pinctrl subsystem to describe peripheral pin mappings.
