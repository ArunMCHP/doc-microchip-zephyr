Acquire a resource mapping for the Xen domain.

Issues the XENMEM_acquire_resource hypercall to map a resource buffer (e.g., I/O request server, grant table, VM trace buffer) into the specified domain's physmap, or query its total size.

Parameters
domidTarget domain identifier. Use DOMID_SELF for the calling domain.
typeResource type identifier (e.g., XENMEM_resource_ioreq_server).
idResource-specific identifier (e.g., server ID or table ID).
frameStarting frame number for mapping, or ignored if *nr_frames == 0.
nr_frames[in,out] On input, number of frames to map; on return, number of frames actually mapped (or total frames if queried).
frame_listGuest frame list buffer: input GFNs for HVM guests, output MFNs for PV guests.
Returns
Zero on success, or a negative errno code on failure.
