espressif,esp32-lpuart
######################

.. sidebar:: Overview

   :Name: ``espressif,esp32-lpuart``
   :Vendor: :ref:`Espressif Systems <dt_vendor_espressif>`
   :Used in: :zephyr:board-catalog:`List of boards <#compatibles=espressif,esp32-lpuart>` using
               this compatible
   :Driver: :zephyr_file:`drivers/serial/lpuart_esp32.c`

Description
***********

These nodes are "uart" bus nodes.

.. code-block:: none

   ESP32 Low Power UART
