sitronix,st7796s (on mipi-dbi bus)
##################################

.. sidebar:: Overview

   :Name: ``sitronix,st7796s``
   :Vendor: :ref:`Sitronix Technology Corporation <dt_vendor_sitronix>`
   :Used in: :zephyr:board-catalog:`List of boards <#compatibles=sitronix,st7796s>` using
               this compatible
   :Driver: :zephyr_file:`drivers/display/display_st7796s.c`

Description
***********



.. code-block:: none

   Sitronix ST7796S Display controller

   The Sitronix ST7796S is a 262K colors TFT-LCD controller
   with a maximum 320RGBx480 resolution.
