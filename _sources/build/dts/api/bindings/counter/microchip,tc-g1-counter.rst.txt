microchip,tc-g1-counter
#######################

.. sidebar:: Overview

   :Name: ``microchip,tc-g1-counter``
   :Vendor: :ref:`Microchip Technology Inc. <dt_vendor_microchip>`
   :Used in: :zephyr:board-catalog:`List of boards <#compatibles=microchip,tc-g1-counter>` using
               this compatible
   :Driver: :zephyr_file:`drivers/counter/counter_mchp_tc_g1.c`

Description
***********



.. code-block:: none

   Microchip TC (Timer Counter)

   Microchip TC (Timer Counter)

   The TC module supports event counting, time interval measurement,
   and periodic interrupt generation.
   The supported devices:
     - tc-u2249 (3.0.0)
   The supported families are:
     - same5xd5x
