Description
***********



.. code-block:: none

   Nordic Wi-Fi network interface.

   This binding defines a Wi-Fi network interface for Nordic nRF70 series
   Wi-Fi chips. The interface is used to identify and name Wi-Fi network
   interfaces like wlan0.
