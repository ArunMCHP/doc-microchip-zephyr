st,stm32-clock-mux
##################

.. sidebar:: Overview

   :Name: ``st,stm32-clock-mux``
   :Vendor: :ref:`STMicroelectronics <dt_vendor_st>`
   :Used in: :zephyr:board-catalog:`List of boards <#compatibles=st,stm32-clock-mux>` using
               this compatible
   :Driver: :zephyr_file:`drivers/clock_control/clock_stm32_mux.c`

Description
***********



.. code-block:: none

   STM32 Clock multiplexer

   Describes a clock multiplexer, such as per_ck on STM32H7 or
   CLK48 on STM32WB.
   The only property of this node is to select a clock input.
   For instance:
           &perck {
                   clocks = <&rcc STM32_SRC_HSI_KER CKPER_SEL(0)>;
                   status = "okay";
           };
