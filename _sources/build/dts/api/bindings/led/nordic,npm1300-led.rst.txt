nordic,npm1300-led
##################

.. sidebar:: Overview

   :Name: ``nordic,npm1300-led``
   :Vendor: :ref:`Nordic Semiconductor <dt_vendor_nordic>`
   :Used in: :zephyr:board-catalog:`List of boards <#compatibles=nordic,npm1300-led>` using
               this compatible
   :Driver: :zephyr_file:`drivers/led/led_npm13xx.c`

Description
***********



.. code-block:: none

   Nordic nPM1300 LED controller

   The nPM1300 has three LED outputs.
   Each LED can automatically display error or charging status,
   or be controlled by software.
