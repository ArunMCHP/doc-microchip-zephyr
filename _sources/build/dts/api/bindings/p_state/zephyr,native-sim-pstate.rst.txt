zephyr,native-sim-pstate
########################

.. sidebar:: Overview

   :Name: ``zephyr,native-sim-pstate``
   :Vendor: :ref:`Zephyr Project <dt_vendor_zephyr>`
   :Used in: :zephyr:board-catalog:`List of boards <#compatibles=zephyr,native-sim-pstate>` using
               this compatible

Description
***********



.. code-block:: none

   Native Sim mock implementation of custom properties for performance state (pstate)

**********

           - ``int``
           - .. code-block:: none

                for the P-state driver to switch to.
