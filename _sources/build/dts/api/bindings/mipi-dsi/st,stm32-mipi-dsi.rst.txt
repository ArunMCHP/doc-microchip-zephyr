st,stm32-mipi-dsi
#################

.. sidebar:: Overview

   :Name: ``st,stm32-mipi-dsi``
   :Vendor: :ref:`STMicroelectronics <dt_vendor_st>`
   :Used in: :zephyr:board-catalog:`List of boards <#compatibles=st,stm32-mipi-dsi>` using
               this compatible
   :Driver: :zephyr_file:`drivers/mipi_dsi/dsi_stm32.c`

Description
***********

These nodes are "mipi-dsi" bus nodes.

.. code-block:: none

   STM32 MIPI DSI host
