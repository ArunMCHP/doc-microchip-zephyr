zephyr,fake-regulator
#####################

.. sidebar:: Overview

   :Name: ``zephyr,fake-regulator``
   :Vendor: :ref:`Zephyr Project <dt_vendor_zephyr>`
   :Used in: :zephyr:board-catalog:`List of boards <#compatibles=zephyr,fake-regulator>` using
               this compatible
   :Driver: :zephyr_file:`drivers/regulator/regulator_fake.c`

Description
***********



.. code-block:: none

   Fake regulator, to be used as a mock or stub in tests.

   An arbitrary number of children can be defined.

Properties
**********

Top level properties
====================

No top-level properties.

          request status of all devices fed by the regulator.

          Contrast with supply-gpios.  If both properties are provided
