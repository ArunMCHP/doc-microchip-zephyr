Detailed Description

SoC and Board hooks.

This header file contains function prototypes for the interfaces between Zephyr's architecture and initialization code and SoC/board-specific logic that resides under boards/ and soc/

Note
These are all standard SoC and board interfaces that are exported from SoC/board-specific logic to OS internal logic. These should never be accessed directly from application code but may be freely used within the OS.
