fixed-layout
############

.. sidebar:: Overview

   :Name: ``fixed-layout``
   :Vendor: :ref:`Generic or vendor-independent <dt_no_vendor>`
   :Used in: :zephyr:board-catalog:`List of boards <#compatibles=fixed-layout>` using
               this compatible

Description
***********



.. code-block:: none

   Fixed layout for Non-Volatile memory.

Properties
**********

nodes themselves. This page also describes child node
properties in the following sections.

Child node properties
=====================

.. list-table::
   :widths: 1 1 4
   :header-rows: 1

   * - Name
     - Type
     - Details

   * - ``reg``
     - ``array``
     - .. code-block:: none

          This should be in the format <OFFSET SIZE>, where OFFSET
          is the offset of the NVMEM cell relative to the base
          address of the parent memory, and SIZE is the size of
          the cell in bytes.



       This property is **required**.

       See :ref:`dt-important-props` for more information.

   * - ``read-only``
     - ``boolean``
     - .. code-block:: none

          Set this property if the cell is read-only.
