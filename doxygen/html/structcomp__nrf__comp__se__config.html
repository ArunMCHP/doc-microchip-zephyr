Detailed Description

Single-ended mode configuration structure.

Note
extrefsel is only used if refsel == COMP_NRF_COMP_REFSEL_AREF
Hysteresis down in volts = ((th_down + 1) / 64) * ref
Hysteresis up in volts = ((th_up + 1) / 64) * ref
