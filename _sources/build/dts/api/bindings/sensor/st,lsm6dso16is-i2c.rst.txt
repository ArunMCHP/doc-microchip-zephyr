st,lsm6dso16is (on i2c bus)
###########################

.. sidebar:: Overview

   :Name: ``st,lsm6dso16is``
   :Vendor: :ref:`STMicroelectronics <dt_vendor_st>`
   :Used in: :zephyr:board-catalog:`List of boards <#compatibles=st,lsm6dso16is>` using
               this compatible
   :Driver: :zephyr_file:`drivers/sensor/st/lsm6dso16is/lsm6dso16is.c`

Description
***********



.. code-block:: none

   STMicroelectronics LSM6DSO16IS 6-axis IMU (Inertial Measurement Unit) sensor
   accessed through I2C bus

Examples
********

.. code-block:: dts

   #include <zephyr/dt-bindings/sensor/lsm6dso16is.h>

   lsm6dso16is: lsm6dso16is@0 {
     /* ... */

     accel-range = <LSM6DSO16IS_DT_FS_8G>;
     accel-odr = <LSM6DSO16IS_DT_ODR_104Hz_LP>;
     gyro-range = <LSM6DSO16IS_DT_FS_2000DPS>;
     gyro-odr = <LSM6DSO16IS_DT_ODR_104Hz_LP>;
   };
