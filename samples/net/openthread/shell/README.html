OpenThread shell

Browse source code on GitHub

Overview

This sample allows testing the Thread protocol and the underlying IEEE 802.15.4 drivers for various boards using the OpenThread shell.

Building and Running

Verify that the board and chip you are targeting provide IEEE 802.15.4 support.

There are configuration files for different boards and setups in the shell directory:

  • prj.conf Generic config file.

  • overlay-ot-rcp-host-nxp.conf This overlay config enables support of OpenThread RCP host running on NXP chips over IMU interface.

Build shell application like this:

west build -b <board to use> samples/net/openthread/shell -- -DCONF_FILE=<config file to use>

Example building for Nordic’s nRF52840 DK.

west build -b nrf52840dk/nrf52840 samples/net/openthread/shell -- -DCONF_FILE="prj.conf"

Example building for NXP’s RW612 FRDM (RCP host).

west build -b frdm_rw612 samples/net/openthread/shell -- -DEXTRA_CONF_FILE="prj-ot-host.conf"

Example building for NXP’s MCXW72 FRDM (host).

west build -b frdm_mcxw72/mcxw727c samples/net/openthread/shell -- -DEXTRA_CONF_FILE="prj-ot-host.conf"

Sample console interaction

uart:~$ ot scan
| PAN  | MAC Address      | Ch | dBm | LQI |
+------+------------------+----+-----+-----+
| fe09 | abcdef1234567890 | 15 | -78 |  60 |
Done
