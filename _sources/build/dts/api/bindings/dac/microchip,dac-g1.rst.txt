microchip,dac-g1
################

.. sidebar:: Overview

   :Name: ``microchip,dac-g1``
   :Vendor: :ref:`Microchip Technology Inc. <dt_vendor_microchip>`
   :Used in: :zephyr:board-catalog:`List of boards <#compatibles=microchip,dac-g1>` using
               this compatible
   :Driver: :zephyr_file:`drivers/dac/dac_mchp_g1.c`

Description
***********



.. code-block:: none

   Microchip G1 DAC

   Microchip G1 DAC peripherals.

   Group G1 DAC includes the following hardware peripherals:
     - module name="DAC" id="U2502" version="1.0.0"

**********

Child node properties
=====================

.. list-table::
   :widths: 1 1 4
   :header-rows: 1

   * - Name
     - Type
     - Details

   * - ``reg``
     - ``array``
     - .. code-block:: none

          DAC channel number.

          Allowed channel numbers are 0 to (max-channels - 1).
          For example, if max-channels = 2, the allowed channels are 0 and 1.



       This property is **required**.

       See :ref:`dt-important-props` for more information.

   * - ``rate``
     - ``int``
     - .. code-block:: none

          Conversion Speed in KSPS(Kilo Samples per second)



       This property is **required**.

       Legal values: ``100``, ``500``, ``1000``

   * - ``ext-filter``
     - ``boolean``
     - .. code-block:: none

          Enable External Filter



   * - ``data-adj``
     - ``string``
     - .. code-block:: none

          Controls how the DAC value is aligned in the DATA register.

          "right" selects right alignment and "left" selects left alignment,
          as controlled by the LEFTADJ bit. The default is "right", which
          matches the hardware reset value of LEFTADJ = 0.
          Right alignment places the LSB of the 12-bit value at bit 0 of DATA.



       Default value: ``right``

       Legal values: ``right``, ``left``

   * - ``dither-mode``
     - ``boolean``
     - .. code-block:: none

          Enable the Dithering Mode



   * - ``sampling-ratio``
     - ``int``
     - .. code-block:: none

          Set the Oversampling Ratio

          Setting oversampling disables the refresh rate.



       Default value: ``1``

       Legal values: ``1``, ``2``, ``4``, ``8``, ``16``, ``32``

   * - ``refresh-period``
     - ``int``
     - .. code-block:: none

          Refresh Period in us ( 0 - Disabled )



       Legal values: ``0``, ``30``, ``60``, ``90``, ``120``, ``150``, ``180``, ``210``, ``240``, ``270``, ``300``, ``330``, ``360``, ``390``, ``420``, ``450``

   * - ``zephyr,resolution``
     - ``int``
     - .. code-block:: none

          DAC resolution to be used for the channel.



   * - ``zephyr,buffered``
     - ``boolean``
     - .. code-block:: none

          Enable output buffer for the channel.



   * - ``zephyr,internal``
     - ``boolean``
     - .. code-block:: none

          Enable internal output path for the channel.



   * - ``zephyr,vref-mv``
     - ``int``
     - .. code-block:: none

          This property can be used to specify the voltage (in millivolts)
          of the reference selected for this channel, so that applications
          can get that value if needed for some calculations.
