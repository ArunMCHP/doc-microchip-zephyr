Pin configuration bit field.

Bitmap:

  • 0-10: Configuration bits (function select, pull, drive strength, etc.)
  • 11: Reserved
  • 12-13: Port (SA, PA, ...)
  • 14-21: Pad (0-128)
  • 22-23: PINR register field (0-3)
  • 24:31: PINR register offset (1-255, 0=not used)
Parameters
portPort
padPad
fselFunction select
pinr_offsetPINR register offset
pinr_fieldPINR register field

Analog mode pin configuration.

Configures a pin for analog mode with FSEL=15, PE=0 (pull disabled), and IE=0 (input disabled). Used for ADC inputs, power supply pins, or other analog functions.
